1. HTML の基礎知識

◆HTML とは

Web ページの構造や、コンテンツの役割を設定する言語です。
HTML で記述したファイルをブラウザに読み込ませる事で、
リンクや画像としてページ上に表示させる事が出来ます。

  • 見出し
  • 文章
  • リンク
  • 画像
  • ボタン など

◆Web サイトの成り立ち

Web サイトは複数の Web ページで構成されます。Web ページは
HTML ファイルを土台とし、CSS や JavaScript、画像データを
関連付けして構築されます。この関連付けを外部参照と呼びます。
今後の学習にも関わるので覚えておきましょう。



◆Web サイトとブラウザ

Web サイトを閲覧する際は、次のしくみが発生します。


  1. 端末(クライアント)からブラウザを立ち上げ、URL の入力やリンクを押す。
  2. ブラウザがサーバ（別のコンピュータ）へリクエストを送信
  3. サーバから Web ページのデータが戻って来る。
  4. ブラウザで Web ページが表示され、閲覧が可能となる。

このように、コンピュータの役割を分担し、ネットワークで連携させたものを
クライアントサーバシステムと言います。クライアント側で機能する
技術をフロントエンドといい、HTML や CSS、JavaScript が活躍します。



◆HTML の書き方

「開始タグ：<>」と「終了タグ：</>」と呼ばれる記号を用いてコンテンツを
囲むように記述します。 下の例では「テキスト」という文字に対し、段落を
指定する「p タグ」で囲んでいます。

<p>テキスト</p>

開始タグから終了タグまでの記述を「要素(element)」といいます。
要素によっては終了タグが無い場合もあります。

4. a タグ

a タグは別のウェブページやファイル、ページ内へのリンクを作成します。このタグでは href 属性を記述し、

2. HTML 入門

1.HTML の基礎知識では、代表的なタグをそれぞれ記述して、HTML の記述に慣れてもらいました。
この章では、Web ページの構造を作るタグを中心に解説し、Web ページを設計していきましょう!


◆HTML の基本構造

Web ページの基本構造は、次のタグを中心に記述します。

!DOCTIPE html
HTMLのバージョンを宣言する部分です。使用するHTMLの規格をブラウザに伝える役割を持ちます。

html タグ
HTML文書であることを宣言するタグで、すべての要素の親となります。つまり記述される全てのタグは、
htmlタグの開始タグと終了タグの間に収まる必要があります。また、言語の属性を記述する場所でもあるため、
日本語のサイトであれば lang="ja"を記述します。

head タグ
文書に関する全般的な情報(メタデータ)を記述します。具体的には文書のタイトルや外部参照先の
リンクなどを配置します。次に解説する meta タグと title タグを含みます。

meta タグ
文書に関する情報を指定します。文字コード charset="utf-8" を記述する場所の認識でOKです。

title タグ
ページのタイトルを指定します。タイトルは、ブラウザのタブ表示される部分です。

body タグ
ページのコンテンツ要素を指定します。前の章で学習した見出しや文章、画像などはここに記述します。

これらのタグをコードとして記述すると、次のようになります。 
<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8" />
    <title>タイトル</title>
  </head>
  <body>
    <p>テキスト</p>
  </body>
</html>

上記コードを見ると、タグの中に別のタグが入った構造が複数出て来ます。この構造をネスト(入れ子)と呼びます。
タグの中に入る要素は、原則としてインデントを入れて字下げし、どのタグの中にあるか分かりやすくします。

◆header タグと footer タグ

上記のタグ以外に、Web ページの構造に関わるタグを 2 つ解説します。


header タグ
サイト内でヘッダー部分(上部)を指定するタグです。サイト名やナビゲーター、記事の見出使われます。

footer タグ
サイト内でフッター部分(下部)を指定するタグです。関連文書へのリンクや、著作権情報等を記載します。

ページ構造は、以下の図のようなイメージを持っていただければ良いです。絶対的なルールではありませんが、
構造を示すタグを記述する事で、構成の整理やこの後学習する CSS にも役立つので、慣れていきましょう。




3. CSS の基礎知識

◆CSS とは

Web ページの文字や色、コンテンツの配置などを整える言語です。近年では HTML はページの構造、
CSS はページのレイアウトと、それぞれ役割を分担する方法が主流となっています。


◆CSS ファイルの外部参照

記述した CSS を Web ページに反映させるためには、(1) 内部参照(HTML ファイルに CSS を記述する) 、
(2) 外部参照 (HTML ファイルから CSS ファイルを読み込む) のいずれかを行いますが、原則は(2)を用います。
(1) だとコードが長くなって可読性が下がり、全体的な修正も手間がかかるためです。




◆link タグ

外部ファイルを指定するタグです。CSS の外部参照などに用いられ、head タグ内に以下の要素を記述します。

<link rel="stylesheet" href="ファイルの参照先" />

rel 属性ではファイルとの関係性(今回は CSS のため、Stylesheet)、href 属性では ファイルの参照先を記述します。
HTML ファイルと CSS ファイルを同じフォルダ(階層)に保存した場合は、ファイル名のみ記述します。

注意して欲しいのが、今回使用している CodePen ではあらかじめ HTML と CSS が連携されているため、
外部参照を記述しなくても、CSS の内容が反映されます。実際に Web ページを 1 から作成する場合は、
必ず link タグによる外部参照を記述しましょう。


◆CSS の書き方

対象となる HTML 要素をセレクタ、指定したい内容をプロパティ、プロパティの詳細を値で記述します。また、セレクタの後には
{} (波括弧) を書き、プロパティと値を記述します。プロパティと値の間には : (コロン)、値の後には ; (セミコロン) を記述します。


上記の例は「p タグ内のコンテンツに、赤色を指定する」という記述になります。


◆ フォントプロパティ

まずはフォントの見た目を指定するプロパティから勉強しましょう。

<p>Apple</p>

p {
  font-style: italic; /* italic, normal, oblique */
  font-weight: bold; /* normal, lighter */
  font-size: 30px;
  font-family: cursive; /* arial, Georgia, sans-serif,  */
}

CSS カラープロパティ値

参照ページを見ると、プロパティ値に「RGB 16 進表記」表示がありました。これについて説明します。
RGB は、Red Green Blue の頭文字で、赤・緑・青 を混合する割合で色を指定します。RGB カラーモデルと
呼ばれる表現法で、0 ～ 255 の数値で表示します。例えば「R:0 G：0 B:0」 の場合は「白」になります。

また、0 ～ 255 の数字を 2 桁の 16 進数(0 ～ 9 と A ～ F までの数字で表す記法)で表現する事も可能です。
そのため、いずれかのプロパティ値で色を指定します。直感的にはキーワード、細かい設定は RGB カラーが
使いやすいです。



◆ ボックスモデル

上記の演習内容を実行すると、テキストの背景色が画面の端まで続く四角である事、そして 2 つのテキストの間に
隙間がある事に気付いたと思います。これは、ボックスモデルと呼ばれるレイアウトが関係しています。

ボックスモデルとは、HTML の各要素がボックスと呼ばれる長方形の領域を生成し、その中にコンテンツや余白、
境界線が含まれる状態になっています。CSS では、要素のコンテンツや余白に対してプロパティを設定し、
位置や幅、色を指定する事が可能になります。
